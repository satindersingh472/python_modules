Modules:
1. Create a new Git/GitHub repo named python_modules
2. Configure your debugger
3. Create a new file app.py
4. Create a new folder helpers
1. Inside helpers create a log.py
5. Inside of log.py, create a function that will take 1 string as an 
argument and print out ’LOGGING: ’ followed by the string
6. Import the log.py file in app.py
7. Call the function in log.py 3 times from app.py
8. Add, commit, and push your code
